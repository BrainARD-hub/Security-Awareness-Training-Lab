# 🎯 GoPhish Phishing Simulation Lab

Because one click can change everything.

This project is a hands-on cybersecurity lab designed to help organizations, students, and instructors **simulate phishing attacks safely** and improve security awareness using **GoPhish**, **MailHog**, and **Kali Linux**.

Instead of waiting for real attackers, this lab allows you to **test, measure, and strengthen human security** in a controlled environment—where mistakes become lessons, not breaches.

---

## 🚀 What You’ll Learn

- How to set up GoPhish for phishing simulations  
- How to use MailHog to safely capture emails locally  
- How to design realistic (but ethical) phishing campaigns  
- How to track user behavior (opens, clicks, reports)  
- How to turn results into effective security awareness training  

---

## 🧰 Tools Used

- GoPhish (Phishing simulation framework)  
- MailHog (Local email testing server)  
- Golang (Required for MailHog)  
- Kali Linux (Lab environment)  

---

## ⚠️ Disclaimer

This project is for **educational and authorized security awareness training only**.

Do NOT use these techniques on real users, systems, or organizations without **explicit permission**. Unauthorized use may be illegal and unethical.

---

## 💡 Why This Lab Matters

Technology alone cannot stop phishing attacks.

People are the first line of defense.

This lab helps you build a culture where users:
- Think before they click  
- Recognize suspicious emails  
- Report threats early  

Because security is not just about tools—  
it’s about awareness.
